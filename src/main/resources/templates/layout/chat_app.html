다들 뭘 하고 있을까요?

    메인화면 대화방 매뉴얼

    0. 로그인한 유저만 대화에 참여 가능
    * guest - 읽기모드

    1. 입력란 컨트롤 - 입력란 활성화 : Enter
    - 줄바꿈 태그 입력 : Shift+Enter
    - 채팅창 스크롤 : PageUp/PageDown
    - 채팅창 처음-끝 스크롤 : (Shift) PageUp/PageDown
    - 이전 채팅 복사 : 위아래 화살표

    2. 공지
    - 새 유저 메인화면 입장시 (입장했습니다)
    - 유저 메인화면 이탈시 (나갔습니다)
    - 신고시 (신고되었습니다)
    - 60초 정지 풀릴시 (신고가 해제되었습니다)
    - 경고 : 유명인 사칭, 혐오표현, 비속어, 장문도배(150자)

    3. 멀티미디어
    - 이미지 확장자(.png, .jpg, .gif) 등으로 끝나는 링크를 올리면 이미지 전송
    - 일반 하이퍼링크를 올리면 링크버튼 활성화

    4. 신고 기능
    - 유저 말풍선 끝에 [신고]를 누르면 해당 유저 60초 채팅 정지
    - 페이지를 벗어나면 해제
    - /free [유저이름] 입력하면 신고 해제

    5. 명령어
    - 대화창 클리어 : /클린 /c /clean /clear
    - 사용자 공지 : /공지 /note /notice [공지내용]
    - 채팅창 왼쪽으로 : /left /l
    - 채팅창 오른쪽으로 : /right /r
    - 채팅창 최소화 : /mini /m /최소화 /숨기기 /hide
    - 유저 신고하기 : /ban [유저이름]
    - 유저 신고풀기 : /free [유저이름]
    - 날씨 확인하기 : /날씨 [날짜] [지역]

    6. 창 컨트롤
    - 말풍선 아이콘 클릭 : 채팅
    - 말풍선 아이콘 드래그 : 채팅창 위치 이동
    - [ ? ] 버튼 : 사용법
    - [ x ] 버튼 : 채팅창 최소화 (읽기 모드)
    - 채팅창 우측 아래 모서리 드래그 : 채팅창 길이 조절
    - 유저 프로필 클릭하면 프로필 확대
